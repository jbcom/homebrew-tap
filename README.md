# jbcom/homebrew-tap

Homebrew tap for [@jbcom](https://github.com/jbcom)'s tools.

## Install

```bash
brew tap jbcom/tap
brew install <formula>
```

## Available formulas

- [`ralph`](Formula/ralph.rb) — [radioactive-ralph](https://github.com/jbcom/radioactive-ralph), autonomous continuous development orchestrator for Claude Code

Formulas are auto-generated and published by GoReleaser from each tool's own repository on release.
